# AnimeCLZ

## Objetivo

AnimeCLZ será una plataforma profesional de streaming de anime.

Debe poder crecer hasta convertirse en un negocio.

No es un proyecto de práctica.

---

## Identidad

Nombre oficial:

AnimeCLZ

Inspiración:

- Crunchyroll
- AniList
- Apple TV
- Dribbble
- Awwwards

No copiar Netflix.

---

## Idioma

Toda la interfaz estará en Español Latino.

El código permanecerá en inglés siguiendo buenas prácticas.

## Objetivos

- Tener un proyecto profesional para portafolio.
- Aprender React.
- Aprender arquitectura profesional.
- Preparar AnimeCLZ para convertirse en un negocio.
- Tener un panel administrador.
- Poder subir mis propios animes.
- Consumir la API de Jikan.
- Mezclar contenido público con contenido propio.
